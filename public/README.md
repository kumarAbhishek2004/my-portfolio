# Add Your Resume and CV Here

## Required Files

Place the following files in this `public/` folder:

1. **resume.pdf** - Your resume document
2. **cv.pdf** - Your CV document

These files will be downloadable from the hero section of your portfolio.

## File Requirements

- **Format:** PDF only
- **File names:** Must be exactly `resume.pdf` and `cv.pdf` (lowercase)
- **Size:** Keep under 5MB for optimal loading
- **Location:** Must be in the `public/` folder (this folder)
